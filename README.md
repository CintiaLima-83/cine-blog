
🎬 CineBlog
Aplicação de blog feita com Next.js 15 (App Router) para resenhas de filmes, Séries/artigos.

📂 Estrutura
app/page.tsx → Página inicial com listagem dos artigos.

app/artigos/[slug]/page.tsx → Página dinâmica para cada artigo.

⚡ Funcionalidades
Listagem de artigos com título, autor, data e resumo.

Página de detalhe com conteúdo completo.

Metadados dinâmicos (title e description) por artigo.

Suporte a SSG com generateStaticParams.

🔧 Tecnologias
Next.js 15

React Server Components

JSON local ou API externa (ex: TMDB, crudcrud)
